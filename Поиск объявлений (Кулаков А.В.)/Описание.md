# Рамки: Приложение электронной доски объявлений.

## Cценарий: Создание корзины

## Уровень: Задача, определенная пользователем

## Основной исполнитель: Зарегистрированный пользователь

### Заинтересованные лица и их требования:

- Покупатель:
  - Хочет добавлять объявления в корзину для последующего оформления заказа
  - Хочет управлять содержимым корзины: просматривать, изменять количество, удалять товары
  - Хочет сохранять корзину между сеансами

- Продавец:
  - Хочет видеть статистику по добавлениям в корзину для аналитики спроса

- Администратор:
  - Хочет отслеживать активность корзин

- Компания:
  - Хочет увеличить конверсию от просмотра к покупке через удобство корзины
  - Хочет анализировать тренды заказов для оптимизации предложений

### Предусловия:

- Система содержит опубликованные и прошедшие модерацию объявления
- Пользователь имеет доступ к интерфейсу поиска
- Пользователь должен быть авторизован

### Результаты (Постусловия):

- Пользователь может перейти к оформлению заказа
- Сохраненная корзина доступна для повторного использования

### Основной успешный сценарий:

1. Пользователь открывает каталог объявлений
2. Пользователь нажимает на "Добавить в корзину" (можно сделать несколько раз)
3. Пользователь переходит во вкладку "Корзина"
4. Пользователь видит список объявлений в корзине
5. Пользователь оформляет заказ
   - _Смотреть прецедент "Оформление заказа"_
6. Корзина очищается

### Расширения (альтернативные потоки):

- Пользователь удаляет товар из корзины
  > 1. Система обновляет результаты в реальном времени (AJAX-подобный запрос).
  > 2. Пользователь видит уточненный список.

- Пользователь выходит из корзины без оформления заказов:
  > 1. Система отображает сообщение "Ничего не найдено" с предложениями: расширить фильтры, проверить опечатки показать похожие объявления.
  > 2. Пользователь может скорректировать запрос и повторить поиск.

### Специальные требования:

- Добавление в корзину должно быть простым и понятным
- Подтверждение добавления объявлений в корзину
- Подтверждение удаления объявлений из корзины
- Подтверждение перехода к оформлению заказа

### Список технологий и типов данных:

- База данных: хранение корзины с объявлениями
- Уведомления: для отслеживания изменений в корзине
- Аналитика: сбор статистики о добавленных объявленияъ в корзину

### Частота использования:

- Высокая: постоянное добавление объявлений в корзину, оформление заказов
